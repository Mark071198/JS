Описание програмнного продукты

Программа для лопания пузырей.

Особенности программного продукта

Управление происходит за счет наведение курсора мыши на объекты и лопание их нажатием левого клика

Есть возможность начать игру снова после истечения данного времени

Детали реализации

Особенности реализации отдельного функционала представлены ниже:

Рандомизация размеров и позиции

Код задает размеры пузырей и устанавливает из изначальное расположение.

                
        var pageWidth = $(document).width();
        $('#bubble-' + this.id).css("left", Math.floor(Math.random() * pageWidth / 100 * 35) + 20);
        var size = Math.floor(Math.random() * 60) + 50 + "px";
        $('#bubble-' + this.id).css("width", size);
        $('#bubble-' + this.id).css("height", size);

Обратный отсчет

Показывает обратный отсчет во время игры

                
        var seconds_left = this.duration / 1000;
        var interval = setInterval(function() {
            $('#timer_div').html(--seconds_left);

            if (seconds_left <= 0) {
                $('#timer_div').html("Time is over"); 
                $('#create').html("Play again?");
                $('#create').show();
                clearInterval(interval);
            }
        }, 1000);

Время между генерацией пузырей

Ожидает случайного количества времени между генерацией пузырей.

                
        for (i = 0; i < this.bubbles; i++) {
            setTimeout(function() {
                var newBubble = new Bubble();
                newBubble.blow();
            }, Math.floor(Math.random() * this.duration - 800) + 0);
